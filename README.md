# Expense Tracker Application

A secure expense tracking system built with **Spring Boot**, **MySQL**, and **JWT**. This application allows users to manage their expenses and categories with role-based authentication and robust REST APIs.

---

## Features

- **JWT Authentication:** Secure login and session management.
- **Expense Management:** Create, read, update, and delete expenses.
- **Category Management:** CRUD operations for expense categories.
- **Validation & Error Handling:** Input validation and global exception handling to ensure API reliability.
- **RESTful APIs:** Fully functional endpoints for integration with frontend applications.

---

## Tech Stack

- **Backend:** Spring Boot
- **Database:** MySQL
- **Authentication:** JWT
- **API:** RESTful CRUD APIs
